## 1. 화면 목록

| ------------------------- | -------------------- | ---------------------------- | --------------------------------------------------------------------------------------------- |
| 로그인                    | `/login`             | 사용자 인증                  | 이메일, 비밀번호 입력, 로그인 버튼, 소셜 로그인 버튼                                          |
| 회원가입                  | `/signup`            | 신규 사용자 계정 생성        | 약관 동의, 이메일/비밀번호 입력, 프로필 설정, 신체 정보 입력                                  |

| 문의 게시판               | `/board`             | 전체 게시글 목록             | 공지글, 게시글 목록, 검색, 페이지네이션, [문의하기] 버튼                                      |

## 2. 전체 화면 흐름

### 2.1 비회원 사용자 플로우

```
시작
  ↓
로그인 화면
├─ 회원가입 클릭 → 회원가입 화면
│ └─ 회원가입 완료 → 메인 홈
└─ 로그인 성공 → 메인 홈
```

### 2.2 로그인 사용자 플로우

```
메인 홈
├─ [날씨 위젯] 클릭 → 날씨 상세 화면 → 메인 홈
├─ [목표 설정] 버튼 → 목표 추천 메인 → 목표 상세 → 목표 진행 → 메인 홈
├─ [기록 보기] 버튼 → 기록 관리 메인
│   ├─ [기록 추가] → 기록 추가 화면 → 기록 관리 메인
│   └─ 기록 카드 클릭 → 기록 상세 → 수정/삭제 → 기록 관리 메인
└─ [프로필] 메뉴 → 마이페이지
    ├─ 프로필 수정 → 마이페이지
    ├─ 신체 정보 수정 → 마이페이지
    └─ 문의 게시판 → 게시글 작성/조회 → 마이페이지
```

### 2.3 화면 전환 다이어그램

```
┌─────────────┐
│  로그인     │
└──────┬──────┘
       ↓
┌──────────────────────────────────────┐
│         메인 홈                       │
│  - 헤더 (로고, 날씨 위젯, 유저 상태)  │
│  - 동기부여 멘트                      │
│  - 목표 설정 / 기록 보기 버튼         │
│  - 마라톤 대회 정보                   │
│  - 하단 네비바 (홈/기록/목표/프로필)  │
└──┬────────┬──────────┬──────────────┘
   ↓        ↓          ↓          ↓
┌─────┐ ┌──────┐ ┌──────┐ ┌──────────┐
│날씨  │ │기록  │ │목표  │ │마이페이지│
│상세  │ │관리  │ │추천  │ │         │
└─────┘ └──────┘ └──────┘ └──────────┘
```

## 3. 홈/로그인/회원가입

### 3.1 로그인 화면

#### 구성 요소

- 앱 로고
- 이메일 입력 필드
- 비밀번호 입력 필드
- 로그인 버튼
- 구글 / 카카오 소셜 로그인 버튼
- 회원가입 이동 링크

#### 사용자 행동

- 이메일/비밀번호 입력 후 로그인 버튼 클릭
- 소셜 로그인 버튼 클릭 → OAuth 인증
- 회원가입 링크 클릭 → 회원가입 화면으로 이동

#### 상태

- **정상**: 입력 필드 활성화, 로그인 버튼 활성화
- **입력 오류**: 해당 필드 하이라이트 및 에러 메시지 표시
- **로그인 실패**: "이메일 또는 비밀번호가 일치하지 않습니다" 메시지
- **로딩 중**: 로그인 버튼 비활성화, 로딩 스피너 표시

#### 화면 이동

- 로그인 성공 → 메인 홈
- 회원가입 클릭 → 회원가입 화면

#### UI 구성

```
┌─────────────────────────┐
│   [로고] 서브5           │
│                         │
│   로그인                │
│                         │
│   이메일                │
│   [input: email]        │
│                         │
│   비밀번호              │
│   [input: password]     │
│                         │
│   [로그인 버튼]          │
│                         │
│   [구글 로그인]          │
│   [카카오 로그인]        │
│                         │
│   회원가입              │
└─────────────────────────┘
```

### 3.2 회원가입 화면

#### 목적

신규 사용자의 계정을 생성한다.

#### 구성 요소

- **1단계: 약관 동의**
  - 필수 약관 체크박스
  - 선택 약관 체크박스
- **2단계: 기본 정보**
  - 이메일 입력 + 중복 확인 버튼
  - 비밀번호 입력
  - 비밀번호 확인 입력
  - 닉네임 입력 + 중복 확인 버튼
- **3단계: 프로필 설정**
  - 프로필 이미지 선택 (선택)
- **4단계: 신체 정보**
  - 성별 선택 (라디오 버튼)
  - 생년월일 선택 (date picker)
  - 키 입력 (cm)
  - 몸무게 입력 (kg)
  - BMI 자동 표시

#### 사용자 행동

- 단계별 정보 입력
- [다음] 버튼 클릭 → 다음 단계 이동
- [가입 완료] 버튼 클릭 → 회원가입 처리

#### 상태

- **필수 항목 미입력**: [다음] 버튼 비활성화
- **이메일 중복**: "이미 사용 중인 이메일입니다" 메시지
- **닉네임 중복**: "이미 사용 중인 닉네임입니다" 메시지
- **비밀번호 규칙 위반**: "비밀번호는 8자 이상, 영문/숫자/특수문자를 포함해야 합니다" 메시지
- **BMI 자동 계산**: 키와 몸무게 입력 시 실시간 계산 및 표시

#### 화면 이동

- 회원가입 완료 → 메인 홈 (자동 로그인)

#### UI 구성

```
┌─────────────────────────┐
│   회원 가입             │
│   [1단계: 약관 동의]     │
│                         │
│   □ 서비스 이용약관 동의 │
│   □ 개인정보 처리방침 동의│
│                         │
│   [다음]                │
└─────────────────────────┘

┌─────────────────────────┐
│   회원 가입             │
│   [2단계: 기본 정보]     │
│                         │
│   이메일                │
│   [input] [중복확인]     │
│                         │
│   비밀번호              │
│   [input]               │
│                         │
│   비밀번호 확인          │
│   [input]               │

### 3.3 메인 홈 화면

#### 구성 요소

- **상단 헤더** (홈 화면에만 고정)
  - 로고
  - 날씨 위젯 (로그인 시에만 표시)
  - 유저 상태 아이콘 (드롭다운)

- **메인 콘텐츠**
  - 동기부여 멘트
  - [목표 설정] 버튼
  - [기록 보기] 버튼
  - 마라톤 대회 정보 카드
  - 국내/해외 탭

- **하단 네비게이션 바** (고정)
  - 홈, 기록, 목표, 프로필

#### 사용자 행동

- 날씨 위젯 클릭 → 날씨 상세 화면
- [목표 설정] 버튼 클릭 → 목표 추천 메인
- [기록 보기] 버튼 클릭 → 기록 관리 메인
- 대회 카드 [신청하기] 버튼 클릭 → 외부 사이트 새 창
- 하단 네비 메뉴 클릭 → 해당 화면 이동

#### 상태

- **날씨 로딩 중**: "날씨 정보를 불러오는 중..." 메시지
- **대회 정보 없음**: "등록된 대회 정보가 없습니다" 메시지
- **신청 URL 없음**: [신청하기] 버튼 비활성화

#### UI 구성

```
┌─────────────────────────────────┐
│ [로고] [18°C ☀️] [유저 아이콘▼]  │ ← 헤더 (홈 화면만)
├─────────────────────────────────┤
│                                 │

│                                 │

│   [기록 보러가기]                │
│                                 │
├─────────────────────────────────┤
│   마라톤 대회 정보               │
│   [국내] [해외]                  │
│                                 │
│   ┌───────────────────────┐    │
│   │ D-30                   │    │
│   │ 서울 마라톤             │    │
│   │ 📍 서울 잠실             │    │

## 4. 실시간 날씨 및 네비게이션

### 4.1 날씨 위젯 (헤더)

#### 구성 요소

- 온도 (18°C)
- 날씨 이모지 (☀️)
- 간단 안내 문구 ("러닝하기 좋은 날씨!")
- 배경 색상 (날씨 상태별)

#### 사용자 행동

- 위젯 클릭 → 날씨 상세 화면으로 이동

#### 상태

- **로딩 중**: "..." 표시
- **데이터 없음**: "날씨 정보 없음"
- **위치 권한 거부**: "위치 권한이 필요합니다"
- **API 오류**: "날씨 정보를 불러올 수 없습니다"

#### UI 구성

```
┌────────────────────────┐
│ 18°C ☀️ 러닝하기 좋은 날씨! │
└────────────────────────┘
```

### 4.2 날씨 상세 화면

#### 구성 요소

- **헤더**
  - ← 뒤로가기 버튼
  - "날씨 정보" 제목

- **위치 정보**
  - 📍 동 단위 주소 (예: "서울시 강남구 역삼동")
  - [위치 재확인] 버튼

- **날씨 정보 카드**
  - 온도: 18°C
  - 습도: 55%
  - 날씨: 맑음
  - 풍속: 2.5 m/s
  - 체감온도: 17°C

- **러닝 적합도 카드**
  - 적합도 배지 (☀️ 최적)
  - 안내 문구 ("러닝하기 완벽한 날씨입니다! 🏃‍♂️")

#### 사용자 행동

- ← 뒤로가기 버튼 클릭 → 메인 홈으로 이동
- [위치 재확인] 버튼 클릭 → 위치 재조회 및 날씨 데이터 갱신

#### 상태

- **위치 재확인 로딩**: "위치를 확인하는 중..." 메시지
- **Geocoding 실패**: 좌표만 표시

#### UI 구성

```
┌─────────────────────────┐
│ [←] 날씨 정보           │
├─────────────────────────┤
│ 📍 서울시 강남구 역삼동   │
│ [위치 재확인]            │
├─────────────────────────┤

## 8. 공통 UI 요소

### 8.1 로딩 상태

- **스켈레톤 UI**: 콘텐츠 영역에 회색 박스 애니메이션
- **스피너**: 중앙에 로딩 아이콘 표시
- **텍스트**: "로딩 중..." 메시지

### 8.2 에러 상태

- **에러 메시지**: 붉은색 텍스트로 에러 내용 표시
- **재시도 버튼**: [다시 시도] 버튼 제공

### 8.3 빈 상태

- **안내 메시지**: "데이터가 없습니다" 또는 구체적인 안내
- **CTA 버튼**: "첫 기록 추가하기" 등

### 8.4 확인 모달

```
┌─────────────────────────┐
│ 확인                    │
├─────────────────────────┤

│                         │
│ [취소] [확인]           │
└─────────────────────────┘
```

┌─────────────────────────┐

```
